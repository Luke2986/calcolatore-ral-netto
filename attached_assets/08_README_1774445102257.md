# Calcolatore RAL → Netto — README

Prototipo di calcolatore retribuzione netta annuale e mensile per un dipendente a tempo indeterminato, costruito come exercise per JetHR.

---

## Cosa fa

Riceve una RAL in input. Restituisce il netto annuale e mensile con il dettaglio di tutte le voci trattenute: contributi INPS, IRPEF (lorda e netta), detrazioni lavoro dipendente, addizionali regionali e comunali, bonus cuneo fiscale, trattamento integrativo.

Anno fiscale: 2026. Caso: impiegato privato, tempo indeterminato, residente a Milano, senza carichi di famiglia, 13 mensilità.

---

## Come usare il prototipo

Aprire `calcolatore_ral_netto.html` nel browser. Inserire una RAL. Cliccare "Calcola".

Non serve nessun server, nessuna installazione, nessuna dipendenza. Il file è autocontenuto.

## Semplificazioni principali

| Semplificazione | Impatto | Motivazione |
|-----------------|---------|-------------|
| Solo impiegato TI, Milano | Non copre altri contratti/residenze | Caso standard richiesto dall'esercizio |
| Trattamento integrativo solo per redditi ≤ €15K | Fascia €15K-€28K non coperta (condizioni complesse) | Trade-off consapevole: la logica per la fascia intermedia richiede detrazioni specifiche non modellate |
| Nessun carico di famiglia | Niente detrazioni figli/coniuge | Da specifica |
| Nessun onere deducibile | Non riduce l'imponibile IRPEF | Caso semplice |
| Tredicesima non calcolata separatamente | Nella realtà ha trattamento specifico per bonus/trattamento integrativo | Semplificazione dichiarata |
| Addizionali calcolate a scaglioni marginali | Il metodo potrebbe variare per regione | Coerente con la fonte (sito Regione Lombardia) |

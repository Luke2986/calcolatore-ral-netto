# Calcolatore RAL → Netto: Documento Logiche e Calcolo

## Contesto e Semplificazioni

**Prodotto:** Prototipo di calcolatore retribuzione netta per JetHR  
**Anno fiscale di riferimento:** 2026  
**Normativa:** Legge di Bilancio 2026 (L. 199/2025), TUIR aggiornato

### Assunzioni del prototipo

| Parametro | Valore assunto | Motivazione |
|-----------|---------------|-------------|
| Tipo contratto | Tempo indeterminato | Da specifica |
| Qualifica | Impiegato settore privato | Da specifica |
| Residenza | Milano (Lombardia) | Da specifica |
| Mensilità | 13 | Standard CCNL |
| Giorni lavoro/anno | 365 | Anno pieno |
| Familiari a carico | Nessuno | Da specifica |
| Altri redditi | Nessuno (reddito complessivo = reddito lavoro dipendente) | Semplificazione |
| TFR | Destinazione INPS/fondo (non impatta il netto mensile) | Semplificazione |
| Benefit in natura | Nessuno | Semplificazione |
| Fondo previdenza complementare | No | Semplificazione |
| Assegno unico figli | Non applicabile | Nessun figlio |

### Semplificazioni consapevoli e loro impatto

1. **Addizionali regionali Lombardia:** calcolate con aliquote progressive (come da delibera regionale). Non a scaglioni marginali come IRPEF, ma per scaglioni sull'intero reddito. *Nota: alcune fonti indicano applicazione progressiva, altre a scaglioni. Il prototipo applica il metodo progressivo per scaglioni (marginale), coerente con quanto riportato dal sito della Regione Lombardia.*

2. **Trattamento integrativo (ex Bonus Renzi):** incluso per i casi di capienza fiscale con reddito ≤ €15,000. Per redditi €15,001-€28,000 le condizioni sono più complesse (dipendono da specifiche detrazioni) e non sono implementate nel prototipo.

3. **Bonus cuneo fiscale 2026:** implementato sia come bonus (reddito ≤ €20,000) sia come ulteriore detrazione (€20,001-€40,000).

4. **Contributo aggiuntivo INPS 1%:** applicato sulla quota di retribuzione eccedente €56.224 annui (prima fascia pensionabile 2026).

## Pipeline di Calcolo

```
RAL (input)
  │
  ├─► [1] Contributi INPS dipendente
  │       = RAL × 9.19% + (RAL - 56,224) × 1% (se RAL > 56,224)
  │
  ├─► [2] Imponibile IRPEF = RAL - contributi INPS
  │
  ├─► [3] IRPEF lorda (scaglioni progressivi)
  │
  ├─► [4] Detrazioni lavoro dipendente (art. 13 TUIR)
  │
  ├─► [5] Ulteriore detrazione cuneo (€20,001-€40,000)
  │
  ├─► [6] IRPEF netta = IRPEF lorda - detrazioni (min 0)
  │
  ├─► [7] Trattamento integrativo (se applicabile)
  │
  ├─► [8] Bonus cuneo fiscale (se reddito ≤ €20,000)
  │
  ├─► [9] Addizionale regionale Lombardia
  │
  ├─► [10] Addizionale comunale Milano
  │
  └─► [OUTPUT] Netto = RAL - INPS - IRPEF netta - add.reg. - add.com.
                        + bonus cuneo + trattamento integrativo
```

## Dettaglio Calcoli

### [1] Contributi INPS a carico del dipendente

**Base normativa:** Aliquota IVS 33% totale, di cui 9.19% a carico lavoratore.

```
contributi_base = RAL × 0.0919

// Contributo aggiuntivo 1% (L. 438/1992)
// Soglia prima fascia pensionabile 2026: €56.224 (Circolare INPS 6/2026)
if (RAL > 56224) {
  contributo_aggiuntivo = (RAL - 56224) × 0.01
}

totale_INPS = contributi_base + contributo_aggiuntivo
```

### [2] Imponibile IRPEF (Reddito complessivo)

```
imponibile = RAL - totale_INPS
```

*Nota: nel caso semplificato, reddito complessivo = imponibile IRPEF = reddito da lavoro dipendente netto dei contributi.*

### [3] IRPEF Lorda — Scaglioni 2026

**Base normativa:** Art. 11 TUIR, come modificato da L. 199/2025 (Legge di Bilancio 2026). Seconda aliquota ridotta dal 35% al 33%.

| Scaglione | Da | A | Aliquota |
|-----------|----|---|----------|
| 1° | €0 | €28,000 | 23% |
| 2° | €28,001 | €50,000 | 33% |
| 3° | €50,001 | — | 43% |

```
function calcolaIRPEFLorda(reddito) {
  if (reddito <= 28000)
    return reddito × 0.23
  if (reddito <= 50000)
    return 28000 × 0.23 + (reddito - 28000) × 0.33
  return 28000 × 0.23 + 22000 × 0.33 + (reddito - 50000) × 0.43
}
// Valori fissi: 28000 × 0.23 = 6,440 | 22000 × 0.33 = 7,260
```

### [4] Detrazioni lavoro dipendente — Art. 13 TUIR

**Base normativa:** Art. 13, comma 1, lettere a), b), c) del TUIR.

Il calcolo usa il **reddito complessivo** (= imponibile IRPEF nel nostro caso) e il rapporto ai giorni lavorati (365/365 = 1 per anno pieno).

```
function detrazioniLavoroDipendente(reddito) {
  if (reddito <= 15000)
    return max(1955, 690)  // minimo 690 per tempo indeterminato

  if (reddito <= 28000)
    return 1910 + 1190 × (28000 - reddito) / 13000

  if (reddito <= 50000) {
    base = 1910 × (50000 - reddito) / 22000
    // Maggiorazione €65 per redditi €25,001-€35,000 (comma 1.1)
    if (reddito > 25000 && reddito <= 35000)
      base += 65
    return base
  }

  return 0  // Nessuna detrazione oltre €50,000
}
```

### [5] Ulteriore detrazione cuneo fiscale (L. 207/2024, art. 1, comma 6)

**Condizione:** Reddito complessivo > €20,000 e ≤ €40,000.  
**Applicazione:** Solo su redditi da lavoro dipendente (no pensioni).

```
function ulterioreDetrazioneCuneo(redditoComplessivo) {
  if (redditoComplessivo > 20000 && redditoComplessivo <= 32000)
    return 1000

  if (redditoComplessivo > 32000 && redditoComplessivo <= 40000)
    return 1000 × (40000 - redditoComplessivo) / 8000

  return 0
}
```

### [6] IRPEF Netta

```
irpef_netta = max(0, irpef_lorda - detrazione_lavoro_dip - ulteriore_detrazione_cuneo)
```

### [7] Trattamento Integrativo (ex Bonus Renzi)

**Base normativa:** Art. 1, DL 3/2020, come modificato da L. 207/2024.

**Importo:** €1,200/anno (€100/mese).  
**Condizione di capienza:** IRPEF lorda (calcolata solo sui redditi lavoro dipendente) > detrazione art. 13 comma 1 diminuita di €75.

```
function trattamentoIntegrativo(redditoComplessivo, irpefLorda, detrazioneLavDip) {
  if (redditoComplessivo > 15000) return 0  // Semplificazione prototipo

  // Verifica capienza (come da circolare 4/2025 AdE, confermata per il 2026)
  soglia_capienza = detrazioneLavDip - 75
  if (irpefLorda > soglia_capienza)
    return 1200

  return 0
}
```

*Nota: per i redditi €15,001-€28,000 il trattamento integrativo ha condizioni aggiuntive (verifica detrazioni specifiche) non implementate nel prototipo.*

### [8] Bonus cuneo fiscale (L. 207/2024, art. 1, comma 4)

**Condizione:** Reddito complessivo ≤ €20,000.  
**Natura:** Somma che NON concorre a reddito imponibile.  
**Calcolo:** Percentuale applicata al reddito da lavoro dipendente.

```
function bonusCuneo(redditoComplessivo, redditoLavoroDip) {
  if (redditoComplessivo > 20000) return 0

  if (redditoLavoroDip <= 8500)
    return redditoLavoroDip × 0.071

  if (redditoLavoroDip <= 15000)
    return redditoLavoroDip × 0.053

  return redditoLavoroDip × 0.048
}
```

*Nota: nel nostro caso semplificato, redditoComplessivo = imponibile IRPEF = reddito lav. dip.*

### [9] Addizionale regionale IRPEF — Lombardia

**Base normativa:** L.R. Lombardia, aliquote 2026 (regime transitorio: scaglioni confermati).  
**Base imponibile:** Reddito complessivo (= imponibile IRPEF).

| Scaglione | Da | A | Aliquota |
|-----------|----|---|----------|
| 1° | €0 | €15,000 | 1.23% |
| 2° | €15,001 | €28,000 | 1.58% |
| 3° | €28,001 | €50,000 | 1.72% |
| 4° | €50,001 | — | 1.73% |

```
function addizionaleRegionale(reddito) {
  if (reddito <= 15000)
    return reddito × 0.0123
  if (reddito <= 28000)
    return 15000 × 0.0123 + (reddito - 15000) × 0.0158
  if (reddito <= 50000)
    return 15000 × 0.0123 + 13000 × 0.0158 + (reddito - 28000) × 0.0172
  return 15000 × 0.0123 + 13000 × 0.0158 + 22000 × 0.0172 +
         (reddito - 50000) × 0.0173
}
```

### [10] Addizionale comunale IRPEF — Milano

**Base normativa:** Deliberazione C.C. Milano n. 36 del 21/10/2013.  
**Aliquota:** 0.8% (unica).  
**Esenzione:** Reddito imponibile ≤ €23,000 → esenzione totale (NON è franchigia).

```
function addizionaleComunale(reddito) {
  if (reddito <= 23000) return 0
  return reddito × 0.008
}
```

### [OUTPUT] Calcolo finale

```
netto_annuale = RAL
  - totale_INPS
  - irpef_netta
  - addizionale_regionale
  - addizionale_comunale
  + bonus_cuneo
  + trattamento_integrativo

netto_mensile = netto_annuale / 13   // su 13 mensilità
netto_mensile_12 = netto_annuale / 12 // media mensile effettiva
```

## Limiti noti del prototipo

1. **Non copre contratti a termine** (detrazione minima diversa: €1,380).
2. **Non gestisce rapporti part-year** (giorni < 365).
3. **Non gestisce part-time** (impatta solo la RAL in input, il calcolo è corretto).
4. **Trattamento integrativo parziale** per redditi €15,001-€28,000: non implementato.
5. **Non considera detrazioni per carichi di famiglia.**
6. **Non considera oneri deducibili** (contributi previdenza complementare, etc.).
7. **Non gestisce la tredicesima separatamente** (nella realtà, trattamento integrativo e bonus cuneo hanno regole specifiche sulla tredicesima).
8. **Addizionali regionali:** il metodo di applicazione (per scaglioni marginali vs. aliquota piena) potrebbe variare. Il prototipo usa il metodo a scaglioni marginali.

# DESIGN SYSTEM — Calcolatore RAL → Netto

Questo documento specifica le scelte visive del prototipo. Deriva dalla sezione 8 del PRD.

Nota: il prototipo è standalone (non integrato in JetHR). Il design è professionale e neutro, non allineato al brand JetHR perché non ne conosciamo il design system interno. Se il tool fosse integrato, adotterebbe il design system della piattaforma.

---

## 1. PALETTE COLORI

```css
/* Sfondo e superfici */
--bg:            #f7f7f5    /* Sfondo pagina — grigio caldo, non bianco puro */
--surface:       #ffffff    /* Card, pannelli */
--surface-alt:   #f0efec    /* Righe alternate, subtotali */
--border:        #e2e0db    /* Bordi card e separatori */

/* Testo */
--text-primary:   #1a1a18   /* Titoli e valori principali */
--text-secondary: #6b6966   /* Label, descrizioni */
--text-muted:     #9c9893   /* Hint, note, percentuali */

/* Accent */
--accent:        #2563eb    /* Blu — CTA, netto evidenziato, card highlight */
--accent-light:  #eff4ff    /* Sfondo card highlight */
--accent-hover:  #1d4fd8    /* Hover bottone */

/* Semantici */
--red:           #dc2626    /* Trattenute (voci negative) */
--red-light:     #fef2f2
--green:         #16a34a    /* Detrazioni e bonus (voci positive) */
--green-light:   #f0fdf4
--orange:        #ea580c    /* INPS nella barra distribuzione */
--yellow:        #f59e0b    /* Addizionale regionale nella barra */
--yellow-light:  #fbbf24    /* Addizionale comunale nella barra */
```

**Logica colore per il breakdown:**
- Voci che tolgono dalla RAL → dot rosso, valore con segno negativo
- Voci che aggiungono (detrazioni, bonus) → dot verde, valore con segno positivo
- Subtotali → sfondo leggermente più scuro, font-weight 600
- Riga finale (netto) → colore accent, font-weight 600, bordo top

## 2. TIPOGRAFIA

| Ruolo | Font | Weight | Size |
|-------|------|--------|------|
| Titolo pagina (h1) | DM Sans | 700 | 28px (desktop), 22px (mobile) |
| Titolo sezione (h3) | DM Sans | 600 | 14px |
| Label campo | DM Sans | 500 | 13px |
| Body / descrizioni | DM Sans | 400 | 14-15px |
| Valori numerici (importi) | DM Mono | 500 | 20-22px (summary), 14px (breakdown) |
| Badge / uppercase | DM Sans | 600 | 11px, letter-spacing 0.5px |
| Note / semplificazioni | DM Sans | 400 | 13px |

**Perché DM Sans + DM Mono:**
- DM Sans: geometric sans-serif, leggibile, tono professionale senza essere corporate
- DM Mono: monospace coerente con DM Sans, perfetto per i numeri (allineamento cifre)
- Entrambi su Google Fonts, zero licensing

**Fallback:** `-apple-system, BlinkMacSystemFont, sans-serif`

## 3. SPAZIATURA E LAYOUT

```
Container:    max-width 880px, padding 24px laterale
Card:         padding 20-28px, border-radius 10px, border 1px solid --border
Gap tra card: 24px (verticale)
Gap tra elementi in card: 12-16px
```

**Responsive breakpoint:** 600px

| Viewport | Layout summary cards | Breakdown | Input |
|----------|---------------------|-----------|-------|
| > 600px | 4 colonne grid | Full-width | Input + bottone inline |
| ≤ 600px | 2 colonne grid | Full-width | Input e bottone stacked |

## 4. COMPONENTI

### 4.1 Input Card
- Card bianca con bordo
- Label sopra il campo ("Retribuzione Annua Lorda")
- Prefisso "€" fisso dentro il campo
- Font monospace per l'input
- Bottone "Calcola" blu accent, affiancato al campo (stacked su mobile)

### 4.2 Summary Cards (4)
- Grid 4 colonne (2 su mobile)
- Ogni card: label uppercase in alto (12px, muted), valore grande al centro (22px, mono), sottotitolo in basso (12px, secondary)
- Le card "Netto Annuale" e "Netto Mensile" hanno sfondo accent-light e bordo accent

### 4.3 Barra Distribuzione
- Barra orizzontale stacked, altezza 36px, border-radius 6px
- Segmenti colorati proporzionali: Netto (blu), INPS (arancione), IRPEF (rosso), Add. Regionale (giallo), Add. Comunale (giallo chiaro)
- Legenda sotto con swatch + label + importo + percentuale

### 4.4 Breakdown Tabellare
- Header con titolo "Dettaglio calcolo"
- Righe con: label a sinistra (con dot colorato), valore a destra (con percentuale)
- Sezioni separate da label uppercase (es. "CONTRIBUTI PREVIDENZIALI", "IRPEF", "ADDIZIONALI")
- Riga subtotale con sfondo diverso
- Riga finale "Netto Annuale" con stile evidenziato
- Righe "Netto Mensile ÷13" e "Media Mensile ÷12" sotto il totale

### 4.5 Note / Semplificazioni
- Card con bordo, sfondo surface
- Titolo bold "Note e semplificazioni"
- Lista puntata con font 13px, colore secondary

### 4.6 Badge "Prototipo"
- In alto prima del titolo
- Sfondo accent-light, testo accent, uppercase, 11px
- Comunica immediatamente che non è un tool di produzione

### 4.7 Assumptions Bar
- Sotto il titolo, prima dell'input
- Sfondo surface con bordo
- Mostra le assunzioni in linea: Contratto, Qualifica, Residenza, Mensilità, Anno, Carichi
- Font 13px

## 5. OMBRE E ELEVAZIONE

```css
--shadow-sm:  0 1px 2px rgba(0,0,0,0.04)        /* Summary cards */
--shadow:     0 1px 3px rgba(0,0,0,0.06),
              0 1px 2px rgba(0,0,0,0.04)          /* Input card, breakdown */
--shadow-lg:  0 4px 12px rgba(0,0,0,0.08)         /* Bottone hover */
```

Ombre minime. Il design è piatto con bordi sottili, non material.
